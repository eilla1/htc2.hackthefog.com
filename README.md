# htc2.hackthefog.com

This is the website for the Hack the Cloud 2.0 hackathon.

## Install dependencies

Use the package manager [npm](https://www.npmjs.com) to install the dependencies and run the site.

```bash
npm install
```

## Run web app

```bash
npm run devStart
```
